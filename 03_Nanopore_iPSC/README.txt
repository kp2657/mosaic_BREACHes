03_Nanopore_iPSC contains scripts and code to start with nanopolish and strique outputs to produce plots in Figure S2.

CGG_count contains the CGG counts for the 7 iPSC-NPCs after guppy.
nanopolish contains the DNA methylation outputs from nanopolish for the 7 iPSC-NPCs after guppy,
strique.txt contains the strique output for the 7 iPSC-NPCs.
DESEQ_melted.txt contains the RNA-seq data from the 7 iPSC-NPCs.

001_Nanopore_SFig1.ipynb produces the plots in S2B,C,E.
